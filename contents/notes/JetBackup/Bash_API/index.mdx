# Instructions

For each function you want to call use the `-F` option:

```bash
jetbackup5api -F listBackups
```

And to pass data use the `-D` option:

```bash
jetbackup5api -F listBackups -D "type=1&contains=511&limit=10"
```

Finally to select output type use the `-O` option:

```bash
jetbackup5api -F listBackups -O plain -D "type=1&contains=511&limit=10"
```
