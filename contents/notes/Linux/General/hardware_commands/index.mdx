# HARDWARE COMMANDS

* lsusb command - shows how many usb there are in the device and information about them.

* lspci command - shows all the buses and devices connected to the system.

* lsblk command - shows all the blocked devices.

* lshw command - it is better to run it as sudo, and the command not always install on the OS you have. it shows all the information about the user hardware.

* lsmod command - shows all the running modules on the system

* rmmod command - removes specific module from the system.

* modprobe - adds specific module to the system.
